# Output Files

This page describes all primary artifacts generated by BirdBox.

## Inference Artifacts (`detect_birds.py`)

### JSON with algorithm metadata (`*.json`)

Top-level keys:

- `audio_file` or `audio_files`
- `model_config`:
  - `model`
  - `confidence_threshold`
  - `nms_iou_threshold`
  - `song_gap_threshold`
  - `species_mapping`
- `detection_count`
- `detections`

Detection keys:

- identifiers: `species`, `species_id`
- confidence: `confidence` (raw) or `avg_confidence`/`max_confidence` (merged)
- geometry: `time_start`, `time_end`, `freq_low_hz`, `freq_high_hz`
- optional multi-file: `filename`, `file_path`, `detections_merged`

### Simplified CSV (`*.csv`)

```text
Filename,Start Time (s),End Time (s),Low Freq (Hz),High Freq (Hz),Species eBird Code,Confidence
```

### Xeno-Canto Annota-JSON (`*.xc.json`)

Export format for Xeno-Canto-compatible workflows.

### Raven selection table (`*.txt` or `*_raven/`)

Tab-separated selection table for Raven.

## F-beta Analysis Artifacts (`f_beta_score_analysis.py`)

- `f{beta}_score_analysis.csv`
- `f{beta}_score_analysis.json`
- `optimal_thresholds.csv`
- plots: micro/macro curves, comparison, per-class curves, heatmap

## Filter-and-Merge Artifacts (`filter_and_merge_detections.py`)

Same output families as inference, plus JSON `filtering_config`:

- `confidence_threshold`
- `song_gap_threshold`

JSON also stores:

- `original_detection_count`
- merged `detection_count`

## Confusion Matrix Artifacts (`confusion_matrix_analysis.py`)

- `confusion_matrix.csv`
- `confusion_matrix_detailed.csv`
- `confusion_matrix_normalized.png`
- `confusion_matrix_raw.png`
- `metadata.txt`
